# 位置詳情頁 (pages/location/[id].vue)

## 頁面功能
- 展示特定潛點的詳細資訊
- 提供潛點相關的服務和資訊

## 主要內容
1. 潛點資訊
   - 潛點名稱
   - 詳細介紹
   - 難度等級
   - 最佳潛水季節
   - 水深範圍
   - 能見度資訊
   - 特色生物

2. 環境資訊
   - 即時天氣
   - 水溫資料
   - 海流狀況
   - 地形特徵
   - 注意事項

3. 相關服務
   - 該潛點可用的潛水服務列表
   - 附近住宿推薦
   - 交通資訊

4. 多媒體內容
   - 潛點照片
   - 水下影片
   - 環境實景

## 互動功能
- 服務預訂連結
- 收藏功能
- 分享功能
- 評論系統
- 潛點評分

## 動態路由
- 使用 [id] 參數載入不同潛點資訊
- 支援所有台灣登記的潛點位置
